# Learning WordPress Theme

This is a **learning-based WordPress theme** created for practice and educational purposes.  
The theme is built to understand **WordPress theme development fundamentals** such as template hierarchy, widgets, custom post types, and custom queries.

---

##  Purpose

- Learn WordPress Theme Structure
- Practice Custom Widgets
- Understand Custom Post Types & Meta Fields
- Work with WP_Query
- Learn widget API (form, update, widget)

---

##  Technologies Used

- PHP
- WordPress Core
- HTML5
- CSS3
- Basic JavaScript (optional)
